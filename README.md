# Academic Personal Website

A clean, minimal academic website combining the typographic elegance of [Brittany Chiang v1](https://bchiang7.github.io/) with the navigation UX of [Brittany Chiang v4](https://v4.brittanychiang.com/).

## Local Development

### Prerequisites

- Ruby ≥ 3.1 ([install guide](https://www.ruby-lang.org/en/documentation/installation/))
- Bundler (`gem install bundler`)

### Steps

```bash
# 1. Clone your fork
git clone https://github.com/yourusername/yourusername.github.io.git
cd yourusername.github.io

# 2. Install gems
bundle install

# 3. Serve locally with live-reload
bundle exec jekyll serve --livereload

# Open http://localhost:4000 in your browser
```
